# v4.1 Demo Runbook (CLI + UI)

Dokumen ini untuk menyiapkan demo singkat v4.1 sambil menunggu rekaman GIF/video final.

## 0) Setup singkat

```bash
npm install
npm run build
```

## 1) Capture CLI transcript

```bash
npm run demo:v41:cli
```

Output transcript disimpan ke:
- `artifacts/demo/v41-cli-demo.txt`

Jika ingin transcript dari benchmark fresh (rebuild + benchmark):

```bash
npm run demo:v41:cli:fresh
```

Isi transcript mencakup:
- `health:summary`,
- `validate:pr5:gaps`,
- snapshot benchmark publik,
- benchmark fresh opsional.

## 2) Demo UI existing Next.js frontend

Gunakan app existing berikut:

```bash
cd examples/standar-config-next-js-app
npm run dev
```

Flow demo UI (30–60 detik):
1. Tunjukkan heading backlog demo.
2. Klik **Add to cart** (state counter berubah).
3. Klik **Switch theme** (live token/theme switching).
4. Ubah lebar viewport untuk menunjukkan container-query grid.
