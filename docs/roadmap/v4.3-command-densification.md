# v4.3 Command Densification

Tujuan v4.3 adalah memadatkan entrypoint command menjadi satu CLI utama: `tw`.

## Ringkasan

- `tw plugin search|list|install`
- `tw create <name> --template=...`
- `tw dashboard --port=...`
- `tw test [--watch]`
- `tw storybook --variants='{...}'`
- `tw code --docs|--install`

## Catatan kompatibilitas

- Command lama (`tailwind-styled`, `create-tailwind-styled`) tetap tersedia.
- `tw` menjadi alias entrypoint baru agar discoverability lebih sederhana.
